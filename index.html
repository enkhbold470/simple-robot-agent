Welcome to Simple Robot Agent

This documentation site provides comprehensive information about the Simple Robot Agent project, a flexible and modular robot control framework built in Python.

Whether you're building a physical robot, simulating robot behavior, or just learning about robotics programming, this library provides a solid foundation for your projects.

Key Features

  • Modular architecture for adding sensors and actuators
  • Task-based operation with a priority queue
  • Clean, well-documented API
  • Simple to use, yet extensible for complex applications
  • Position and orientation tracking
  • Simulation capabilities for testing without hardware

Quick Start

Get started quickly with the following simple example:

from robot_agent import RobotAgent

# Initialize robot
robot = RobotAgent("MyRobot")

# Add components
robot.add_sensor("distance", "ultrasonic")
robot.add_actuator("motor", "servo")

# Add tasks
robot.add_task("move_forward")
robot.add_task("scan_surroundings")

# Run the robot
robot.run()
