Installation

To get started with the Simple Robot Agent, follow these steps:

1. Clone the Repository

git clone https://github.com/enkhbold470/simple-robot-agent.git
cd simple-robot-agent

2. Install Requirements

pip install -r requirements.txt

Basic Usage

Here's a simple example to create and run a robot:

from robot_agent import RobotAgent

# Initialize robot
robot = RobotAgent("MyRobot")

# Add sensors
robot.add_sensor("distance", "ultrasonic", {"max_range": 400})
robot.add_sensor("temperature", "thermometer")

# Add actuators
robot.add_actuator("left_motor", "dc_motor", {"max_speed": 100})
robot.add_actuator("right_motor", "dc_motor", {"max_speed": 100})

# Add tasks
robot.add_task("move_forward_10")
robot.add_task("turn_right_90")
robot.add_task("read_sensors")

# Run the robot
robot.run()

Understanding the RobotAgent Class

The RobotAgent class is the core of the project. Here are the main methods you'll use:

Initialization

robot = RobotAgent(name="SimpleBot")

Creates a new robot agent with the given name.

Adding Sensors

robot.add_sensor(name, sensor_type, config=None)

Adds a sensor to the robot with a name, type, and optional configuration.

Adding Actuators

robot.add_actuator(name, actuator_type, config=None)

Adds an actuator to the robot with a name, type, and optional configuration.

Reading Sensors

value = robot.read_sensor(name)

Reads and returns data from the specified sensor.

Controlling Actuators

robot.set_actuator(name, state)

Sets the state of the specified actuator.

Movement

robot.move(distance)
robot.rotate(angle)

Moves the robot forward by a distance or rotates by an angle (in degrees).

Tasks

robot.add_task(task)
robot.execute_next_task()
robot.run()

Add tasks to the queue, execute the next task, or run all tasks in the queue.
